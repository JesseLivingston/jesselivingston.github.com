Add Scala Library to Sbt Project

Nov 18th, 2013 | Comments

Add scala librarys to sbt project

I create a sbt code, add a sbt eclipse plugin to build.sbt, and turn the project to a eclipse project.

In eclipse, I add the the scala libraries(under my scala home) to the project, I write some actor code use the scala-actors.jar library, and it works fine in eclipse.

But when I compile the code in sbt, it complains missing some classes. Seems I need to add these libraries for sbt in build.sbt or project/Build.scala.

Solution: add below code to build.sbt

libraryDependencies <++= (scalaVersion)(sv =>
    Seq(
        "org.scala-lang" % "scala-library" % sv,
        "org.scala-lang" % "scala-swing" % sv,
        "org.scala-lang" % "scala-actors" % sv,
        "org.scala-lang" % "scala-reflect" % sv
    )
)

Of course I defined scalaVersion to 2.10.3 in build.sbt, after that, invoke update action in sbt, then compile the code, it should work fine too.
